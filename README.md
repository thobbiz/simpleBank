A bank Api made with Golang and postgreSQL, deployed with AWS(taken down since AWS wanna make me bankrupt), it also uses redis for caching and asynq for background workers.

To Run
- Clone the repo using git clone [https://](https://github.com/thobbiz/simpleBank.git)
- Move to the project's root dir and run 'go run .' in the terminal
- Check the swagger endpoint for the documentation and send your requests using postman or curl
